# Password Salting Demo (C)
    This project demonstrates how password salting works using simple salting technique in C.
# Disclaimer
    I made this project only for educational purposes.
    It does NOT use cryptographically secure hashing and must NOT be used in real applications.
# Features 
    * User Registration
    * Password Salting
    * Hash based login verification
    * File based storage
# How to Run
    -> Compile the program using a C compiler (eg. gcc).
    -> Run the executable from the terminal.
# Learning Outcome
    This project helped me understand file handling, structured program design and applying C programming concepts to a real-life problem.
